# 🧩 Rubik's Coding Lab

An interactive, 3D web-based sandbox designed to help developers visualize algorithm logic, permutations, and group theory through a Rubik's Cube interface. 

Built with React 19, Tailwind CSS, Motion, and Lucide icons.

---

## 🚀 Live Demo & Deployment

The application is configured to run locally or deploy instantly to Vercel:

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

---

## ✨ Features

*   **Tactile 3D Cube Canvas**: Drag and swipe to rotate the cube in 3D space with smooth momentum transitions.
*   **Viewport Lock**: Lock rotation to standard 2D face projections for focused moves.
*   **Move History & Undo**: Track your rotation history and undo previous moves with full state restoration.
*   **Algorithms & Scrambling**: Generate random scrambles to test your solving patterns.
*   **Victory Fanfare**: Confetti overlay triggered automatically upon reaching the solved state.
*   **Modern Design**: Sleek, immersive dark-mode interface with clean controls optimized for both desktop and mobile viewports.

---

## 🗺️ Vision: Interactive Learning

The project is designed to evolve into an educational platform where users can learn software architecture, data structures (3D matrices, permutation groups), and algorithm logic using Rubik's Cube states. Future expansions include:
*   Interactive coding tutorials to program cube solving steps.
*   Multiplayer learner rooms.
*   Additional algorithm visualizers (e.g., Interactive 3D Chess).
