## 2024-05-24 - Added tooltips to disabled buttons\n**Learning:** Users can easily get stuck when buttons are disabled without context. Adding a native `title` attribute to dynamically explain the disabled state (e.g., "Select a column first" instead of just "Spin up") dramatically improves discoverability without adding new UI elements or dependencies.\n**Action:** Always provide context for why interactive elements are disabled, especially in complex applications like a 3D sandbox.
